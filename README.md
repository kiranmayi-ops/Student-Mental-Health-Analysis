# Student Mental Health Analysis 

##  About the Project

This is a simple data analysis project that explores student mental health during online learning. The analysis focuses on factors such as screen time, sleep duration, physical activity, and exam anxiety across different education levels.

##  Tools Used

- Python
- Pandas
- NumPy
- Matplotlib

##  Analysis Performed

- Cleaned the dataset by removing records with insufficient data.
- Created an **Anxiety Score** from exam anxiety responses.
- Analyzed:
  - Average Screen Time
  - Average Sleep Duration
  - Average Exam Anxiety
  - Average Physical Activity
- Compared results across **Education Levels** and **Gender**.
- Visualized findings using bar charts and a dashboard.

##  Output

The project generates:

- A **2×2 dashboard** showing:
  - Average Screen Time
  - Average Sleep Duration
  - Average Exam Anxiety
  - Average Physical Activity
- A bar chart showing **student distribution by education level**.
